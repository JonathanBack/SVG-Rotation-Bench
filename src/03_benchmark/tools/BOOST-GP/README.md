# BOOST-GP

**Repository:** https://github.com/MinZhe/BOOST-GP  
**Method:** Bayesian mOdeling Of Spatial Transcriptomics via Gaussian Process  
**Reference:** Li et al. (2021)

## Download & setup

```bash
cd src/03_benchmark/tools/BOOST-GP
# Download as ZIP or clone:
git clone https://github.com/MinZhe/BOOST-GP.git .
```

The main script is `R/boost.gp.R` — this will be `source()`'d by the benchmark script.

## Dependencies

R packages: `Rcpp`, `RcppArmadillo`, `RcppDist`, `MASS`
