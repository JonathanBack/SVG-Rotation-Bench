# SMASH

**Repository:** https://github.com/sealx017/SMASH-package  
**Method:** Scalable Method for Analyzing Spatial Heterogeneity of genes  
**Reference:** Seal, Bitler & Ghosh (2023), PLOS Genetics

## Download & setup

```bash
cd src/03_benchmark/tools/SMASH
# Download as ZIP or clone:
git clone https://github.com/sealx017/SMASH-package.git .
```

The SMASH module is imported via `import SMASH` from the `SMASH/` subdirectory.

## Dependencies

Python packages: `numpy`, `scipy`, `pandas`, `statsmodels`, `matplotlib`
